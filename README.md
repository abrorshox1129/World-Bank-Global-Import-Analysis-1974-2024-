# World Bank — Global Import Tahlili | 1974–2024

**Dunyo tijorati va import tendentsiyalarini tahlil qiluvchi interaktiv dashboard**

## 📊 Loyihaning Maqsadi

Ushbu loyiha 1974-2024 yillari oralig'ida Jahon Banki ma'lumotlarini asosida global import ko'rsatkichlarini tahlil qiladi. Dashboard quyidagi asosiy metrikalarnini namoyish etadi:

- **560T** - Jami import (Trillion $)
- **217** - Mamlakat soni
- **560.45K** - Import Milliard
- **1974–2024** - Yillar Diapazoni
- **29.85** - Bosh Foiz

## 📈 Dashboard Komponenti

### 1. Git'a bilan Jami Import
- Kontinentlar bo'yicha import tahlili
- Kontinentlar: Europe, Asia, North America, South America, Africa, Oceania
- Har bir kontinentning qisman ko'rsatkichi

### 2. Raqamad Guruhi bilan Median Import
- Daromad guruhiga asosida import tahlili:
  - **High Income** - Yuqori daromadli mamlakatlari
  - **Upper Middle Income** - O'rta yuqori daromadli
  - **Lower Middle Income** - O'rta pastroq daromadli
  - **Low Income** - Pastroq daromadli

### 3. Global Import Trendi (1974-2024)
- 50 yillik vaqt oralig'ida import o'sish tendentsiyasi
- Asosiy o'sish nuqtalari va o'zgarishlari
- Yillik statistika va prognozlar

### 4. Rivojlanish va Rivojlanayotgan
- Donut grafigi format
- Developed (Taraqqiyotga erishgan) va Developing (Rivojlanayotgan) mamlakatlar
- Import qiymatining percentli taqsimoti

## 🛠️ Texnologiya

- **Power BI / Excel Dashboard** - Asosiy tahlil instrumenti

- **Interactive Visualizations** - Dinamik grafiklar
